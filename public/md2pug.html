Markdown to Pug CLI tool
v1.1.2
Github
npm

Ez az első node package-em. Létrejöttének oka, hogy szükségem volt egy eszközre, amivel gyorsan publikálhatok markdown fájlokat a honlapomra, aminek forráskódját Pug-ban írtam.

Maga a converter már létezett (markdown-to-pug ), én és egy barátom csak felhasználtuk azt, és egy CLI tool-t csináltunk, hogy könyebben lehessen fordítani a terminlból.
